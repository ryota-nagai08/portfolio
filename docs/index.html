チャットアプリ

使用技術： Java Spring Boot PostgreSQL JavaScript Thymeleaf

研修課題として開発したWebアプリケーションです。
カレンダーからのスケジュール登録や管理者側からのお知らせ登録、チャット機能などを実装しました。
また、チャット機能は各掲示板ごとにメッセージ送信を行う形で設計しました。

  • SpringSecurityによるログイン認証
  • WebSocketを用いたリアルタイムチャット機能
  • FullCalenderライブラリを用いて、カレンダーからのスケジュール登録機能を実装
